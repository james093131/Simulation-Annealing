Using C++ to implement SA


Type the following command to compile the program:
```cpp
make
```
Type the following command to run the program:
```cpp
./SA [Run] [Iteration] [Dimension] [Temperature] [Cool]
```

For example
```cpp
./SA  51 1000 100 1.0 0.95

```
